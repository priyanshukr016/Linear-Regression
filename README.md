# Linear-Regression
This project demonstrates a simple Linear Regression model built using Python to predict a studentâ€™s placement package based on their CGPA. The dataset is loaded and analyzed using Pandas, numerical operations are handled with NumPy, and data visualization is done using Matplotlib.
